# 递归

>  `num + add(num - 1);`递归：自己调用自己

```java
 public static int add(int num){
       if(num==1){
           return 1;
       }
        return num + add(num - 1);
    }
```

> - 如果没有`if`判断，会出现==栈溢出==`StackOverflowError`
> - 没有终止条件，会一直递归下去

# 折半查找（二分查找）

> ==二分查找的前提是：数组已经排好序==

```java
public class BinarySearch {

    public static void main(String[] args){

        int[] a = {1,3,5,7,9,11,13,15,17};
        binarySearch(a, 1);
     }

    private static int binarySearch(int[] a, int key){
        int count = 0;
        int low = 0;
        int high = a.length-1;
        int mid;


        while(low <= high){
            mid = (low + high)/2;
            count++;
            if(a[mid] == key){
                System.out.println("比较次数：" + count);
                return mid;
            }else if(a[mid] > key){
                high = mid -1;
            }else{
                low = mid + 1;
            }
        }
        return -1;
    }
}

```



# 冒泡排序

```java
package cn.tedu;

import java.util.Arrays;

public class Demo2 {


    public static void main(String[] args) {
        int[] num={43,24,6,3,7};
        for (int i = num.length-1; i > 0; i--) {
            for (int j = 0; j < i; j++) {
                if(num[j]>num[j+1]){
                    int temp = num[j];
                    num[j] = num[j + 1];
                    num[j + 1] = temp;
                }
            }
        }
        System.out.println(Arrays.toString(num));
    }
}

```



# 选择排序

> 第一个数依次和后面的数比较

```java
package cn.tedu;

import java.util.Arrays;

public class Demo2 {


    public static void main(String[] args) {
        int[] num={43,24,6,3,7};
        for (int i = 0; i < num.length - 1; i++) {
            for (int j = i+1 ; j < num.length ; j++) {
                if(num[i]>num[j]){
                    int temp = num[i];
                    num[i] = num[j];
                    num[j] = temp;
                }
            }
        }
        System.out.println(Arrays.toString(num));
    }
}

```
